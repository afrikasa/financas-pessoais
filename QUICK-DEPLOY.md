# ⚡ DEPLOY RÁPIDO - 5 MINUTOS

## 🚀 OPÇÃO 1: VERCEL CLI (MAIS FÁCIL)

### **1. Instala Vercel:**
```bash
npm install -g vercel
```

### **2. Faz Deploy:**
```bash
DEPLOY.bat
```

### **3. Primeiro deploy pede login:**
- Escolhe GitHub/Email
- Confirma no browser
- Nome do projeto: financas-pessoais
- Aguarda 2 minutos...

### **4. PRONTO!**
```
✅ https://financas-pessoais-abc.vercel.app
```

**COPIA O URL E USA NO TELEMÓVEL!** 📱

---

## 🌐 OPÇÃO 2: VERCEL WEBSITE (SEM INSTALAR NADA)

### **1. Build local:**
```bash
BUILD.bat
```

### **2. Vai a Vercel.com:**
```
https://vercel.com
→ Sign Up (GitHub)
→ New Project
```

### **3. Upload:**
```
→ "Import from folder"
→ Escolhe pasta "dist/"
→ Deploy!
```

### **4. PRONTO!**
```
✅ URL aparece
```
