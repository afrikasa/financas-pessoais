## APP-FINANCAS-PESSOAIS v1.0.0

**Data:** 2026-03-08  
**Deploy:** https://financas-pessoais-two.vercel.app

### 🎯 Destaques

Esta é a **primeira versão completa** da app de gestão financeira pessoal!

### ✨ Novas Funcionalidades

- ✅ **Dashboard Completo** - Visão geral das finanças
- ✅ **Gráficos Interativos** - Recharts com 3 gráficos diferentes
- ✅ **Dark Mode** - Tema escuro completo
- ✅ **Export PDF** - Relatórios profissionais
- ✅ **PWA** - Instalável no telemóvel
- ✅ **Responsive Design** - Funciona em qualquer dispositivo

### 🔧 Correções

- 🔧 Dark mode nos empréstimos
- 🔧 Gráficos Recharts JSX errors
- 🔧 Vite config para ngrok

### 🛠️ Tech Stack

- React 18.2.0
- Vite 5.0.8
- Tailwind CSS 3.4.0
- Recharts 2.10.3
- jsPDF 2.5.1

### 📱 Como Instalar

1. Acede a https://financas-pessoais-two.vercel.app
2. Menu → "Adicionar ao ecrã inicial"
3. Usa como app nativa!

### 🚀 Como Usar Localmente

```bash
git clone https://github.com/afrikasa/financas-pessoais.git
cd financas-pessoais
npm install
npm run dev
```
