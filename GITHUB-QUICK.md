# ⚡ GITHUB - GUIA RÁPIDO 5 MINUTOS

## 🎯 3 PASSOS SIMPLES:

---

### **PASSO 1: Setup Git (2 min)**

```bash
GIT-SETUP.bat
```

- Digite teu nome
- Digite teu email
- PRONTO!

---

### **PASSO 2: Criar Repo GitHub (2 min)**

1. Vai a: **https://github.com/new**
2. **Repository name:** `financas-pessoais`
3. **Public** ✅
4. **Create repository** (botão verde)
5. **COPIA O URL** que aparece!

Exemplo:
```
https://github.com/marcus-costa/financas-pessoais.git
```

---

### **PASSO 3: Push (1 min)**

```bash
GITHUB-PUSH.bat
```

- Cola o URL copiado
- ENTER
- Aguarda...
- **PRONTO!** Código no GitHub! 🎉

---

## 🔄 ATUALIZAR DEPOIS:

```bash
GITHUB-UPDATE.bat
```

- Digite mensagem (ex: "Fix bug")
- ENTER
- Push automático!

---

## 🚀 BONUS: Auto-Deploy Vercel

### **Conectar GitHub → Vercel:**

```
1. Vercel.com → Teu projeto
2. Settings → Git
3. Connect Repository
4. Escolhe: financas-pessoais
```

### **Resultado:**

```
git push
  ↓
GitHub atualizado
  ↓
Vercel deploy AUTOMÁTICO! 🚀
```

**Nunca mais precisas de DEPLOY.bat!**

---

## 📋 COMANDOS CRIADOS:

| Script | O que faz |
|--------|-----------|
| `GIT-SETUP.bat` | Inicializa Git (1ª vez) |
| `GITHUB-PUSH.bat` | Envia para GitHub (1ª vez) |
| `GITHUB-UPDATE.bat` | Atualiza GitHub |

## ❓ PROBLEMAS?

**"git: command not found"**
→ Instala: https://git-scm.com

**"Permission denied"**
→ Usa Personal Access Token como password
